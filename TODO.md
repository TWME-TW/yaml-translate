## 💡 開發提示

```bash
# 開發時使用虛擬環境
python -m venv venv
source venv/bin/activate  # Windows: venv\Scripts\activate

# 安裝開發依賴
pip install -r requirements.txt
pip install -e .

# 運行測試
pytest

# 代碼格式化
black yaml_translator/

# 代碼檢查
flake8 yaml_translator/

# 類型檢查
mypy yaml_translator/
```
